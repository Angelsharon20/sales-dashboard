# Sales & Revenue Analysis Dashboard

## Features
- KPI cards
- CSV upload
- Region/category/product/date filters
- Monthly revenue trend
- Revenue by category
- Product and region charts
- Top 5 products
- Sales target progress
- AI-style insights
- Download filtered CSV

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

Place your dataset at `data/sales_data.csv` or upload it using the sidebar.
